Tableau Dashboard

Data from: Maven Analytics.


About

For this project, I'll assume the role of a data analyst for a fictional pizza place in New Jersey. I will help the restaurant use data to improve operations. Here are the questions that I will be trying to answer:
    1. What days and times do we tend to be busiest?
    2. How many pizzas are we making during peak periods?
    3. What are our best and worst selling pizzas?
    4. What's our average order value?

Methodology

For this project, I set up the data model using SQL to include all the related instances in one single table. I cleaned and organized the data using Excel, and used Tableau to visualize trends and patterns by creating a dashboard.
